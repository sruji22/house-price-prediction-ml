# House Price Prediction (Machine Learning)

## Overview
This project predicts house prices using supervised machine learning.  
It demonstrates a complete end-to-end workflow including data preparation, model training, and evaluation.

## Tech Stack
- Python
- pandas, numpy
- scikit-learn
- matplotlib
- Jupyter Notebook

## Workflow
- Data loading and preparation
- Train/test split
- Baseline model: Linear Regression
- Improved model: Random Forest Regressor
- Evaluation using RMSE (Root Mean Squared Error)
